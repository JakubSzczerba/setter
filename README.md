# Exchange
> Application for save and update one row.

## Technologies
* PHP - version 8.2.12
* Symfony - version 6.3.7
* PostgreSQL - version 13.2-1

## Local Setup
```
docker compose up -d
```
```
docker compose exec php composer install
```
```
docker compose exec php bin/console doctrine:migrations:migrate
```
